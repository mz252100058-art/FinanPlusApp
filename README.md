- FinanPlus

Aplicación móvil desarrollada en Android Studio utilizando Kotlin y Firebase Firestore para el control financiero personal.

-- Descripción

FinanPlus es una aplicación enfocada en la gestión básica de finanzas personales, permitiendo registrar ingresos y gastos mediante una interfaz moderna en modo oscuro.

El proyecto fue desarrollado como práctica de desarrollo móvil, integrando Firebase y diseño UI/UX en Android Studio.

---

-- Funcionalidades implementadas

✅ Registro de ingresos

✅ Registro de gastos

✅ Actualización automática del saldo

✅ Almacenamiento de datos en Firebase Firestore

✅ Interfaz moderna en tema oscuro

✅ Menú flotante interactivo

✅ Navegación entre pantallas

✅ Integración con RecyclerView

---

-- Funcionalidades en desarrollo

🔄 Mejoras en la pantalla de registros

🔄 Gestión avanzada del perfil

🔄 Eliminación y edición de transacciones

🔄 Estadísticas financieras
